OTETO Project

This is the main Project I'm working at for now. It is a Telegram Bot for your own VPN.

Project aim: allow to install your own VPN network on server, rented by you, in one click.

Features:

— set up permissions to your VPN, so you can share access to it with friends or relatives, if you prefer;

— set up permissions to your server - bot offers to protect it from various attacks and vulnerabilities in 24 ways.

Project is under pre-alpha development at the moment, and the alpha release is planned in this month already!

ControlMicTray Project

This project created for personal purposes, but development was started under educational ones.

Project aim: allow you to control your microphone mute state by assigning hotkey for this event.

Features and limitations:

— tray icon allows to see what is current state of your microphone;

— talkie-walkie mode supported: just turn it on in the app tray menu and set hotkey;

— app supports windows only.

In future releases app will be translated to various languages, and notification support will come.

TimeSoft Project

This is my first pet-project, that was created with my university classmates on second year of studying.

Project aim: recording time expenditure, monitoring it during a given period and visualizing time expenditure for various types of activities.

Featues:

— add, edit and delete your tasks and plans in desktop app;

— telegram bot synchronizes with your desktop account - features are the same as in desktop version;

— visualize your activities in desktop app by watching graphs that it created.
